# 温湿度計を作ってみる(IoT農業)

本プロジェクトではproject3で作成した温湿度を測定するデバイスをインターネット環境下して使用できるようにアップデートしたものになります。
こちらは完成形になります。

![](../images/prototype/prototype4/prototype4.jpg#center)

## デバイスについて

### 追加機能
- データベースの保存
- データの可視化
- csv形式データのダウンロード
