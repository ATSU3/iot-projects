## Supabase
Supabaseとは、オープンソースのPostgreSQLのBaaS(Backend as a Service)です。
バックエンド機能をアプリケーションサーバー側が行ってくれるとても便利なサービスです。

[SupabaseのURL](https://supabase.com/)
